# Userlist Claude Plugins

![Userlist Claude Plugins](https://userlist.com/optimized/assets/illustrations/newsletter-examples-saas.jpg?q=100)

A collection of SaaS email marketing related Claude Code plugins by [Userlist](https://userlist.com).

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [**atomic-emails**](https://github.com/userlist/atomic-emails) | A structured, five-step process for creating SaaS email campaigns without staring at a blank page. |

## Installation and usage

Install the marketplace into Claude Code by running:

```sh
/plugin marketplace add userlist/userlist-claude
```

Afterwards you can install the individual plugins:

```sh
/plugin install atomic-emails@userlist-claude
```

## What is Userlist?

[![Userlist](https://userlist.com/images/external/userlist-logo-github.svg)](https://userlist.com/)

[Userlist](https://userlist.com/) allows you to onboard and engage your SaaS users with targeted behavior-based campaigns using email or in-app messages.

Userlist was started in 2017 as an alternative to bulky enterprise messaging tools. We believe that running SaaS products should be more enjoyable. Learn more [about us](https://userlist.com/about-us/).
